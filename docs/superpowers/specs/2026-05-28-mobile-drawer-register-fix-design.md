# 移动端抽屉面板 + 注册默认团队 设计

## 概述

修复两个问题：(1) 新用户注册后自动加入影视飓风团队，(2) 移动端恢复完整 3D 大脑功能，底部面板改为可滑动抽屉。

## 问题 1：注册自动加入影视飓风

### 后端：AuthService.java

注册成功后自动插入 `user_team` 记录 (userId, teamId=1)。

### 前端：MyTeams 页面

若 `/api/teams/my-teams` 返回空列表，显示影视飓风作为默认团队卡片（纯前端兜底）。

## 问题 2：移动端恢复完整大脑 + 底部抽屉

### 核心思路

- 移除 MiniBrain 回退，移动端和桌面端使用相同的 BrainPointCloud 组件
- 保留已有的移动端性能优化（像素比 2、球体 8 段、星空 500、WebGL 容错）
- 桌面端面板保持原有绝对定位不变
- 移动端面板改为底部抽屉

### 移动端布局

```
┌──────────────────────┐
│                      │
│   3D 脑图（全屏）     │  ← 星空背景，完整交互
│                      │
│                      │
├──────────────────────┤
│  ═══ 手柄 ═══        │  ← 可上拉
│ ◉ ○                 │  ← 左右滑动指示
│ 控制面板 ｜ 节点列表  │  ← 默认占下 1/5，上拉展到 1/2
└──────────────────────┘
```

- 底部手柄条，上拉展开面板
- 左右滑动切换两个面板（控制面板 / 节点列表）
- 桌面端（>=768px）保持原有 `absolute` 定位不变

### 性能优化（已保留，不新增降级）

| 参数 | 桌面端 | 移动端 |
|------|--------|--------|
| 像素比 | 1 (设备默认) | Math.min(devicePixelRatio, 2) |
| 球体段数 | 16 | 8 |
| 星空粒子 | 4000 | 500 |
| WebGL 容错 | 有 | 有 |

### 改动文件

| 文件 | 改动 |
|------|------|
| `backend/.../service/AuthService.java` | 注册后插入 user_team(1L) |
| `frontend/src/pages/Index.jsx` | 移除 MiniBrain 分支，始终渲染 BrainPointCloud |
| `frontend/src/components/BrainPointCloud.jsx` | 移动端面板从 absolute 改为底部抽屉 |

## 验证

1. 新用户注册后，「我的团队」应显示影视飓风
2. 手机访问首页：3D 大脑完整显示（星空背景、可旋转缩放、可点击节点）
3. 底部抽屉可上拉展开、左右滑动切换
4. 桌面端功能不受影响
